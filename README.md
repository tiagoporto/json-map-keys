# object-value-to-object-path ![Open Source Love](https://badges.frapsoft.com/os/v3/open-source.svg)

Replace json values with the key path.

Example:

```json
{
  "a": {
    "b": "Text Note"
  }
}
```

Will generate

```json
{
  "a": {
    "b": "a.b"
  }
}
```
